# Amazla

Landing page for Amazla — Tesla control from your Amazfit watch.

- **Dev:** `npm run dev`
- **Build:** `npm run build`
- **Preview:** `npm run preview`
- After editing logos in root: `npm run sync-logos` (copies to `public/images/`)
- **Deploy (without Wrangler):** use Cloudflare Pages Git integration with:
  - Build command: `npm run build`
  - Build output directory: `dist`
