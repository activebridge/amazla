# Amazla landing (GitHub Pages + Jekyll)

This branch hosts the Amazla landing as a static site from repository root.

## Local run

```bash
bundle install
bundle exec jekyll serve
```

Site is available at `http://127.0.0.1:4000`.

## Deployment

Deploy via GitHub Pages using **Deploy from a branch**:

- Branch: `landing-site`
- Folder: `/ (root)`

The custom domain is configured via `CNAME` (`amazla.app`).
