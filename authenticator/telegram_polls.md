## 📊 Система реакцій

**2 реакції на матч:**
1. Емоджі переможця
2. Емоджі рахунку

**Рахунок:**
```
💯 = 5:0  (ідеально)
🍾 = 5:1  (святкування)
🎉 = 5:2  (перемога)
🤝 = 5:3  (боротьба)
💔 = 5:4  (на межі)
```

**Гравці:**

## 📊 Як рахувати результат

**Після групи дивимось реакції:**

| Реакція | Що означає |
|---------|------------|
| 🕊️ ×4 | Сторчак виграв 4 матчі |
| 💯 ×2 | 2 матчі закінчились 5:0 |

**Партії рахуємо так:**
- 💯 (5:0) = +5 партій переможцю
- 🍾 (5:1) = +5 переможцю, +1 програвшему
- 🎉 (5:2) = +5 переможцю, +2 програвшему
- 🤝 (5:3) = +5 переможцю, +3 програвшему
- 💔 (5:4) = +5 переможцю, +4 програвшему

**Місця = сума виграних партій!**
